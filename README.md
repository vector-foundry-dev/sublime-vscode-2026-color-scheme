# VS Code 2026 Dark - Sublime Text

The modern VS Code 2026 Dark aesthetic, ported cleanly to Sublime Text.

*Note: This package strictly provides a `.sublime-color-scheme` to ensure maximum stability across operating systems. It does not force UI theme engine overrides.*

**Tested on:** Sublime Text Build 4200+

## Installation
1. Open the Command Palette (`Cmd/Ctrl + Shift + P`)
2. Choose **Package Control: Install Package**
3. Search for **VS Code 2026 Dark Color Scheme**

## Recommended Settings
To achieve the authentic VS Code feel, open your user settings (`Preferences > Settings`) and add these overrides:

```json
{
    "color_scheme": "VS Code 2026 Dark.sublime-color-scheme",
    "highlight_line": true,
    "draw_white_space": "selection",
    "caret_style": "solid"
}
```
