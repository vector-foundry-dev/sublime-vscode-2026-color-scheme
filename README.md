# VS Code 2026 Color Scheme - Sublime Text

<img src="https://github.com/user-attachments/assets/1ae488a1-4e29-49d4-8d63-a14a8f1c2598" width="100%" />

The modern VS Code 2026 aesthetic — both Dark and Light variants — ported cleanly to Sublime Text.

*Note: This package strictly provides `.sublime-color-scheme` files to ensure maximum stability across operating systems. It does not force UI theme engine overrides.*

**Tested on:** Sublime Text Build 4200+

## Installation
1. Open the Command Palette (`Cmd/Ctrl + Shift + P`)
2. Choose **Package Control: Install Package**
3. Search for **VS Code 2026 Color Scheme**

## Recommended Settings
To achieve the authentic VS Code feel, open your user settings (`Preferences > Settings`) and add these overrides.

**Dark variant:**
```json
{
    "color_scheme": "VS Code 2026 Dark.sublime-color-scheme",
    "highlight_line": true,
    "draw_white_space": "selection",
    "caret_style": "solid"
}
```

**Light variant:**
```json
{
    "color_scheme": "VS Code 2026 Light.sublime-color-scheme",
    "highlight_line": true,
    "draw_white_space": "selection",
    "caret_style": "solid"
}
```
